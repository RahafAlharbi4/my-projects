📊 COVID-19 Financial Impact on Canadian Families (Power BI Project)
Overview

This project analyzes the financial impact of the COVID-19 pandemic on Canadian households using data visualization and business intelligence tools.

The analysis focuses on changes in income, expenses, and consumption patterns before and after COVID-19 across different demographic groups and regions.

The interactive dashboard was developed using Microsoft Power BI.

Problem Statement

The COVID-19 pandemic significantly affected household finances, leading to:

Income instability
Changes in spending behavior
Increased financial pressure on families
Uneven impact across age groups and regions

This project aims to understand these changes using data-driven insights.

Dataset Description

The dataset includes household financial and demographic variables:

Family Size
Age Group
Area (Urban / Suburban / Rural)
Income Before COVID-19
Income After COVID-19
Expenses Before COVID-19
Expenses After COVID-19
Data Preparation

The following steps were applied:

Handling missing values
Removing duplicate records
Standardizing categories (age groups, regions)
Handling outliers
Ensuring balanced demographic representation
Key Insights
Significant drop in income during COVID-19 period
Increased essential spending (food, housing, healthcare)
Urban households were more affected
Middle-aged groups (45–64) faced the highest financial pressure
Shift in spending behavior toward essential needs
Visualizations Used in Power BI
📈 Line Chart: Income and expenses trends over time
📊 Bar Chart (Age Group): Financial differences across age groups
🌍 Bar Chart (Geography): Regional comparison of financial impact
📦 Stacked Column Chart: Household spending breakdown by category

Conclusion

The analysis shows that COVID-19 had a major impact on household financial stability in Canada, especially in urban areas and middle-aged groups.

The findings highlight the importance of:

Financial support programs
Economic resilience strategies
Data-driven decision making for policymakers
